  /* 大屏幕样式 */

### 移动端适配原则
1. **触摸友好**: 按钮最小44px，间距充足
2. **单手操作**: 重要功能放在拇指可达区域
3. **简化导航**: 使用汉堡菜单，减少层级
4. **内容优先**: 突出核心功能，隐藏次要信息
5. **加载优化**: 优先加载关键内容
